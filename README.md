### 👤 نبذة عني | About Me
أنا خريجة علوم حاسب، شغوفة ببناء وتطوير البرمجيات والحلول الذكية. أركز على تنظيم وإدارة وتوثيق المشاريع البرمجية بكفاءة عالية، وتحويل الأفكار إلى واجهات مستخدم متميزة.

- 🚀 أعمل حالياً على: تطوير مهاراتي البرمجية والمشاركة في بناء نماذج أولية مبتكرة.
- 📚 أتعلم وتعمق في: تقنيات الذكاء الاصطناعي وتطبيقات الهواتف الذكية.
- 💬 يمكنك سؤالي عن: إدارة المشاريع البرمجية، كتابة وثائق المتطلبات (SRS)، وتصميم واجهات المستخدم.
